# SmartCart 🛒

**A responsive, vanilla JavaScript e-commerce frontend.**

---

## 🌟 Overview
SmartCart-JS is a lightweight e-commerce product page built to demonstrate modern web development practices. It features a dynamic product grid, a functional shopping cart, and real-time price calculations, all while maintaining a clean and professional user interface.

## 🚀 Key Features
* **Dynamic Product Rendering**: Generates product cards directly from JavaScript objects.
* **Interactive Shopping Cart**: Allows users to add items and view an updated count in the header.
* **Real-time Totals**: Automatically calculates the total order value using the `.reduce()` method.
* **Localized Currency**: All pricing is displayed in **Indian Rupees (₹)**.
* **Responsive UI**: Optimized for different screen sizes using CSS Grid and Flexbox.

## 🛠️ Tech Stack
* **HTML5**: Semantic structure for accessibility and SEO.
* **CSS3**: Custom styling, CSS variables, and responsive layouts.
* **JavaScript (ES6+)**: DOM manipulation, array methods, and state management.
